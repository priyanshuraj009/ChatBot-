# ChatBot
Popup ChatBot
MedVoice Bot is a simple web-based popup chatbot interface that allows users to interact with a chatbot for medical-related inquiries. This project is built using HTML, CSS, and JavaScript.

Features
Popup Chat Interface: Opens as a popup when clicking the "Open Chat" button.
Chatbot Interaction: Users can type messages and receive responses from the chatbot.
Responsive Design: Designed to work well on various screen sizes.
Usage
Opening the Chatbot:

Click the "Open Chat" button to open the chatbot popup.
Chatting with the Bot:

Type your message in the input field and press "Enter" or click "Send" to send the message.
The chatbot will respond with appropriate messages.
Closing the Chatbot:

Click the "Close" button (the 'x' icon) in the chat header to close the chatbot popup.
